Class MsgEditValidator

A validator can edit its delegate information, such as moniker, website, commission rate, etc.

You must use special or sentinel values to inform that you want to leave the current field untouched. For Description,` you should start with MsgEditValidator.DESC_DO_NOT_MODIFY and change each field you wish to modify individually.
